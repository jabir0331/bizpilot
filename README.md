# BizPilot

A shop management platform for small to mid-scale grocery businesses.
Replaces paper-based tracking with digital inventory, sales, 
purchasing, debt tracking, and profit reporting.

## Tech Stack
- Backend: Spring Boot, PostgreSQL
- Frontend: NextJs
- Architecture: Modular Monolith

## Modules
- Inventory (FIFO lot-based stock tracking)
- Sales (cash and credit)
- Purchasing (cash and credit)
- Ledger (customer and supplier debt)
- Expenses
- Reports
